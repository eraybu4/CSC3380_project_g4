# Geaux Quest: Baton Rouge Travel Application 

{Azaria Green, Njeri Dukes, Kendrick Washington, Amoy Russell, Eli Rayburn, Nicholas Quintero}

***Project Overview***

Geaux Quest is a web-based exploration and rewards application designed around the Baton Rouge metro area.
Users can:
- Create an account
- Log in securely
- Explore local attractions, parks, historical sites
- Earn stars for visiting sites
- Redeem stars for rewards

The app is implemented using a Flask backend, routed HTML Templates, and component based front-end layout using CSS and JavaScript

***Frontend***
- HTML
- CSS
- JavaScript
  
***Backend***
- Python
- Flask Web Framework
- Flask-SQLAlechemy
- Werkzeug Security (password hashing)
  
***Database***
- SQLite (instance/Database.db)
  
***Runtime***
- Any web browser
- No emulator required

***How to Run:***
- install python dependencies: "pip install flask flask_sqlalchemy"
- run Flask application: "python app.py"
- in terminal you should see: "Running on http://127.0.0.1:5000/"
- follow this link to open application
